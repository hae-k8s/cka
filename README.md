### 📚 스터디 진행 방식

- 정기 모임은 매주 화요일 15:00 ~ 16:00
- 매주 지정된 범위의 강의 내용을 Markdown 으로 요약하여 Pull Request 작성
- 매주 사다리타기로 발표자 정하기
- 해당 주의 발표자는 다음 주 회의실 예약하기
- 불참 시, 불참자가 발표자에게 스타벅스 기프티콘 N잔 보내기 (N=누적불참횟수, 경조사 제외)
